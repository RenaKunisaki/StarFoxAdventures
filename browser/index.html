Star Fox Adventures data browser

This app lets you explore the contents of a Star Fox Adventures ISO file and/or save file.

ISO File

Select a Star Fox Adventures ISO file to examine it.

Supported versions:

  • US version 1.0 (MD5: afb0306454b581e37a62170fdfd1de09)
  • Other versions (less information available)
If no file is selected, you can still examine a save file, but not as much information will be available.

Selected file:

None

Save File

Select a Star Fox Adventures save file to examine it.

Supported formats:

  • GCI
  • Raw binary (savegame/saveN.bin)

Selected file:

None
No files are uploaded to the server. You must provide your own legally-obtained copy.
